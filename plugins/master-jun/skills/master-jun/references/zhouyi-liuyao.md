## 梅花 time casting default

- Year branch numbers: 子1、丑2、寅3、卯4、辰5、巳6、午7、未8、申9、酉10、戌11、亥12.
- Double-hour uses the same branch numbers.
- Trigram remainder: 1乾、2兑、3离、4震、5巽、6坎、7艮、0坤.
- Upper trigram = year + lunar month + lunar day, modulo 8.
- Lower trigram = upper sum + hour number, modulo 8.
- Moving line = total sum, modulo 6; remainder 0 means line 6.
- State that other lineages use different formulas.
