# Master Jun

An auditable Eastern-divination skill and Codex plugin. It combines verified natal context with Da Liuren, Xiao Liuren, Zhouyi, Meihua, Bazi, and Zi Wei workflows, exposes its casting basis, handles uncertain times, and freezes formal readings against hindsight rewriting.

## Install in Codex

Add this repository as a plugin marketplace, then install the plugin:

```bash
codex plugin marketplace add <github-owner>/master-jun
codex plugin add master-jun@master-jun
```

Open a new Codex task and address it as `大师`, or invoke `$master-jun` explicitly.

## Download

Use GitHub's **Code → Download ZIP** button. After extracting the archive, Codex can also add the extracted repository folder as a local marketplace.

## Scope

This project treats divination as a traditional interpretive practice rather than scientific proof. No personal profiles, consultation transcripts, or real-world case records are included.
